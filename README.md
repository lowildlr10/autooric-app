## Installation
- Run this command
```
npm install
```

## Create a .env file

- If .env doesn't exist, create the .env file and input this variable
```
NEXT_PUBLIC_API_BASE_URL=http://backend-api.test
```
- Change the http://backend-api.test into the actual backend URL of AutoORIC

## Test the system
- Run this command
```
npm run dev

or

npm run build

npm run start
```
